# PR Comment Templates

## Language Detection

Detect language from these signals (in priority order):
1. Explicit `--lang pt|en` flag
2. PR body language: look for Portuguese indicators (`alteração`, `correção`, `implementação`, `adicionado`, `removido`, `atualizado`, `funcionalidade`, `melhoria`, `está`, `são`, `não`, `também`, `será`)
3. Default: English

## Issue Row Format

Each finding should be formatted as a table row:

```
| {severity_emoji} {severity} | `{file_path}` | L{line} | {description} | {agent_name} |
```

**Severity emojis:**
- Critical: `🔴`
- High: `🟠`
- Medium: `🟡`
- Low: `🔵`
- Info: `⚪`

**Severity mapping (PT-BR):**
- Critical → Crítico
- High → Alto
- Medium → Médio
- Low → Baixo
- Info → Info

## Deduplication Rules

When multiple agents report the same or overlapping issue:
1. **Same file + same line (±5 lines):** Merge into single row, list all source agents
2. **Same concept, different files:** Keep separate but group together
3. **Conflicting recommendations:** Present both with agents attributed, note the disagreement
4. **Priority when merging:** Use the highest severity reported by any agent

## Sorting Order

1. Severity: Critical → High → Medium → Low → Info
2. Within same severity: Sort by file path alphabetically
3. Within same file: Sort by line number ascending
